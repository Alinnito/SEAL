
# Шифрование SEAL

SEAL (англ. Software-optimized Encryption Algorithm, программно-оптимизированный алгоритм шифрования) — симметричный поточный алгоритм шифрования данных, оптимизированный для программной реализации. SEAL позиционируется как увеличивающая длину псевдослучайная функция, используемая для реализации поточного шифрования. Для работы ему требуются восемь 32-битовых регистров и память объемом несколько Kбайт. На предварительном этапе SEAL преобразует ключевую информацию в набор таблиц T, S, R.

# Реализация

Алгоритм состоит из нескольких функций.

Функция sha1_G предварительно обрабатывает ключ a на основе процедуры алгоритма SHA-1. Ключ a разбирается на 5 32-битных слова, объединенных в H. Создается 16 32-битных слов W[0]...W[15] = 0. W расширяется до 80 слов по 32 разряда в каждом. Для t >= 16: W[t] = (W[t-3] ^ W[t-8] ^ W[t-14] ^ W[t-16] << 1). Затем выполняется 80 раундов алгоритма. В результате получаем 160-битное значение по 5 32-битных слов.

Функция word_H нужна для построения таблиц. Она вызывается в другой функции - build_tables, которая заполняет таблицы значениями, получившиеся в word_H. После этого этапа получаем 3 таблицы: T[512], S[256], R[256].

Перед генерацией псевдослучайной функции нужно подготовить четыре 32-битовых служебных регистра A, B, C, D и четыре 32-битовых слова n1, n2, n3, n4. Их значения определяются из таблиц R и T, 32-битового числа n и некоторого числа l в функции registers_init.

Функция SEAL генерирует псевдослучайную последовательность. Процесс шифрования состоит из большого числа итераций, каждая из которых завершается генерацией псевдослучайной функции. Количество пройденных итераций показывает счетчик l. Все они подразделяются на несколько этапов с похожими операциями.

# Использование

Программа имеет графический интерфейс, который имеет 3 текстовых поля. Первые текстовое поле "Исходный текст" предназначено для вставки текста, который необходимо зашифровать. После нажатия на кнопку "Шифрование" во второй текстовом поле "Зашифрованный текст" отобразится исходный текст, зашифрованный с помощью алгоритма SEAL. Чтобы расшифровать текст, нужно нажать на кнопку "Расшифровка", тогда на третьем текстовом поле "Расшифрованный текст" появится расшифрованный текст. 

После перезапуска программа генерируется новый ключ, поэтому зашифрованный текст может меняться при каждом запуске программы.
